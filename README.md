# Notepad (Tkinter)

A simple desktop notepad application built with Python and Tkinter.

## Features

- New file
- Open file
- Save file
- Cut, Copy, Paste
- Basic text editing in a resizable window

## Requirements

- Python 3.
- Tkinter (usually included with standard Python installations)

## Run

From the project folder:

```bash
python notepad.py
```
